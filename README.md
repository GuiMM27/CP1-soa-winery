# 🍷 Checkpoint SOA - Web Service SOAP (Winery)

Este repositório contém a entrega da atividade de Arquitetura Orientada a Serviços (SOA), implementando a publicação e o consumo de serviços SOAP em Java.

## 📂 Estrutura do Projeto

* **WinerySys (Publisher):** O servidor que publica os serviços `WineStockService` (porta 8085) e `WineWarningService` (porta 8086).
* **WineStockClient (Consumer 1):** Cliente configurado para ler o WSDL e consumir o método de consulta do menu.
* **WineOrderClient (Consumer 2):** Cliente configurado para realizar o pedido e consultar o sistema de avisos de estoque.
